# 智能排班智能体 · 报价计算器

参考 Excel「收费计算器 + 成本利润分析」设计的 Web 版报价平台。

## 报价模式（三部分）

| 部分 | 计费方式 | 说明 |
|------|----------|------|
| 企业建模实施费 | 一次性 | 数据建模、系统对接、业务梳理、部署、培训等明细项 |
| 数字员工订阅费 | 按门店/月 | 按门店数分 4 档定价，支持年付 83 折 |
| 增值服务费 | 按月（可选） | 扩展模块、专人服务、维保响应等 |

## 快速启动

```bash
cd agent-quote-platform
node server.js
```

- 报价页：http://localhost:3456
- 管理后台：http://localhost:3456/admin/login.html
- 默认密码：`admin123`

## 验证示例（对齐 Excel 安三胖案例）

输入：150 门店 · 2 年合同 · 2 套对接 · 启用专人企业服务 + 维保响应

| 项目 | 金额 |
|------|------|
| 建模实施费 | ¥61,000 |
| 月订阅（档2 × 79） | ¥11,850/月 |
| 合同期订阅 | ¥284,400 |
| 增值（合同期） | ¥12,000 |
| **合同总金额** | **¥357,400** |
| 首年预算参考 | ¥209,200 |

## 管理后台功能

- 报价单管理与分项明细
- 定价规则（实施费、分档、年付折扣）
- 增值项月费配置
- 成本项配置
- 成本利润分析（毛利率、实施/订阅阶段利润率、盈亏平衡月数）

## 数据文件

- `data/pricing.json` — 实施费明细 + 订阅分档
- `data/addons.json` — 增值项（月费）
- `data/costs.json` — 成本项（后台利润分析用）
- `data/quotes.json` — 客户报价申请

## 技术栈

Node.js 原生 HTTP · HTML/CSS/JS · JSON 存储
