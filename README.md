# 数字员工报价平台

多智能体组合报价计算器，参考 Excel「收费计算器 + 成本利润分析」设计。

## 报价模式

每个智能体均包含三部分：

| 部分 | 计费方式 | 说明 |
|------|----------|------|
| 企业建模实施费 | 一次性 | 数据建模、系统对接、业务梳理、部署、培训 |
| 数字员工订阅费 | 按门店/月 | 5 档分档定价，支持年付 88 折 |
| 增值服务费 | 按月（可选） | 扩展模块、专人服务、维保等 |

**多选智能体**时可享受组合折扣（作用于订阅费合同期合计）：

| 选中数量 | 折扣 |
|----------|------|
| 2 个 | 95 折 |
| 3 个 | 92 折 |
| 4 个 | 90 折 |
| 5 个 | 88 折 |
| 6 个（全 suite） | 85 折 |

## 快速启动

```bash
cd agent-quote-platform
node server.js
```

- 报价页：http://localhost:3456
- 管理后台：http://localhost:3456/admin/login.html
- 默认密码：`admin123`

## 管理后台功能

- 报价单管理与对客报价单文档
- 智能体目录管理（新建/启用/配置）
- 平台设置（组合折扣规则）
- 各智能体独立：定价规则 / 增值项 / 成本配置
- 成本利润分析（多智能体汇总）
